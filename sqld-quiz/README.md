# SQLD 인터랙티브 웹 퀴즈 (로컬)

자동 출제(랜덤/템플릿) → 즉시 채점 → 오답 해설/오답노트 → 학습 통계(LocalStorage 저장)까지 되는 **단일 페이지 웹앱**입니다.

## 실행 방법 (권장)

### 1) 그냥 열기 (가장 쉬움)
- `sqld-quiz/index.html` 파일을 **브라우저로 더블클릭**해서 여세요.

> 이 프로젝트는 `fetch`/모듈 로딩이 없어서 `file://`로 열어도 동작합니다.

### 2) 로컬 서버로 열기 (선택)
터미널에서 아래처럼 실행 후, 브라우저에서 `http://localhost:5173` 접속:

```bash
cd "/Users/nawonoh/SQLD 커서/sqld-quiz"
python3 -m http.server 5173
```

## 주요 기능
- **출제 설정**: 문항 수(5~50), 난이도/영역 필터, 오답 우선/새문항 우선
- **문제 타입**: 객관식 + 빈칸(키워드 입력)
- **즉시 채점**: 제출 즉시 정오답/정답 키워드 표시
- **오답 풀이**: 해설(요점 정리) 제공, 결과 화면에서 오답/미답 일괄 복습
- **오답노트**: 틀린 유형은 다음 퀴즈에서 오답 우선으로 자동 출제
- **통계**: 누적 풀이/정답률/오답노트 수 + 영역별 취약도(3문항 이상 풀면 표시)
- **이어하기**: 세션 저장 후 이어풀기
- **결과 내보내기**: JSON으로 다운로드

## 파일 구성
- `index.html`: UI 뼈대
- `styles.css`: 스타일
- `questionBank.js`: 문제은행 + 템플릿 자동 생성
- `app.js`: 진행/채점/오답노트/통계/세션 저장 로직

## 커스터마이즈(문제 추가)
`questionBank.js`의 `BASE_QUESTIONS` 배열에 문항을 추가하면 됩니다.
